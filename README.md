# Cyber Blog API

Backend-приложение для публикации постов и комментариев с системой валидации и разграничением прав доступа.

## Технологии
* **Python 3.12+**
* **Django 6.0+**
* **Django Rest Framework**
* **PostgreSQL**
* **drf-spectacular** (Swagger/OpenAPI UI)

## Ключевой функционал
- **Пользователи:** Кастомная модель на базе `AbstractUser` с авторизацией и валидацией доменов почты.
- **Посты:** Полноценный CRUD. Автоматическая проверка возраста автора (18+) и фильтрация "запрещенных слов" в заголовках.
- **Комментарии:** Возможность комментировать посты с привязкой к текущему пользователю.
- **Оптимизация:** Применение `prefetch_related` во ViewSets для минимизации нагрузки на базу данных (защита от N+1).

## Права доступа
- **Публичный доступ:** Чтение постов и комментариев.
- **Авторизованные пользователи:** Создание контента.
- **Владельцы/Админы:** Редактирование и удаление своего контента.
- **Только Админ:** Удаление пользователей.

## Установка и запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/NikaDeveloper/cyber_blog
   cd cyber_blog 
   ```
   
2. **Настройте окружение:**

    - Создайте файл .env на основе env_example.

    - Создайте и активируйте виртуальное окружение.
    
    - Установите зависимости:

    ```bash
    pip install -r requirements.txt
    ```
    - Примените миграции:

    ```bash
    python manage.py migrate
    ```
    - Создайте суперпользователя:
    
    ```bash
    python manage.py createsuperuser
    ```
    - Запустите сервер:
    
    ```bash
    python manage.py runserver
    ```
